## 领奖工具

![license](https://img.shields.io/github/license/mashape/apistatus.svg?maxAge=2592000) &nbsp; ![code-style-standard](https://img.shields.io/badge/code%20style-standard-brightgreen.svg)

> #### 一套方便日常活动奖品领取的后台管理系统

**技术栈**

* react
* react-router
* react-spring
* unstated
* event-bus

**特点**

* 日常活动的创建、删除、保存与展示
* 历史活动的查看与编辑
* 获奖名单的上传、活动具体数据的分页展示与导出
* 通过微信公众号推送领奖通知信息
* 生成领奖二维码方便线下扫码领取奖品
